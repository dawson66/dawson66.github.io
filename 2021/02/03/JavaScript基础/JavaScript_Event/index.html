JavaScript之事件对象Event

详情请参考MDN

HTML DOM事件

HTML DOM事件允许Javascript在HTML文档元素中注册不同事件处理程序。

事件通常与函数结合使用，函数不会在事件发生前被执行！（如用户点击）。

事件分类

  • 鼠标事件
  • 键盘事件
  • 框架/对象（Frame/Object）事件
  • 表单事件
  • 剪贴板事件
  • 打印事件
  • 拖动事件
  • 多媒体（Media）事件
  • 动画事件
  • 过渡事件
  • 其他事件

Event属性

  • Event.type：事件的类型，不区分大小写。
  • Event.timeStamp：事件创建时的时间戳（精度为毫秒）。按照规范，这个时间戳时Unix纪元起经过的毫秒数，但实际上，在不同的浏览器中，对此时间戳的定义也有所不同。另外，规范正在将其修改为DOMHighResTImeStamp。
  • Event.target：MDN–对事件原始目标的引用，这里的原始目标指最初派发（dispatch）事件时指定的目标。菜鸟教程–触发事件的源组件（事件注册/绑定所在的组件）。
  • Event.currentTarget：事件触发的当前事件（当前事件，可能是触发事件的源组件，可能是触发的事件组件，即触发事件源组件的子元素。）
  • Event.detail：
  • Event.cancelBubble：Event.Propagation()的历史别名。在事件处理函数返回之前，将此属性的值设置为true，亦可阻止事件继续冒泡。
