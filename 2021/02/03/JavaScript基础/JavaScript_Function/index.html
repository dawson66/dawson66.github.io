立即执行函数和它的小秘密

IIFE（立即调用函数表达式）是一个在定义时就会立即执行的JavaScript函数。 MDN

  • IIFE -Immediately Invoked Funtion Expression.
  • 立即执行函数——也可以理解为：初始化函数（当然，初始化函数不一定要用立即执行函数，但是最好这么用）。
  • 自动执行，执行完成后立即释放。

对于立即执行函数只需记住两点：

  • 页面一加载，它就自己执行了。
  • 执行完成后，会立即释放。

函数的经典必考方法
call

call()方法使用一个指定的this值和单独给出的一个或多个参数来调用一个函数。MDN

描述：

  • call()允许为不同的对象分配和调用属于一个对象的函数/方法。
  • call()提供新的this值给当前调用的函数/方法。你可以使用call来实现继承：写一个方法，然后让另一个新的对象来继承它（而不是在新对象中再写一次这个方法）。
apply

apply()方法于call()方法唯一的区别就是传入的参数不同，call()接收的是参数列表，而apply()接收的是一个参数数组。

callee

arguments.callee属性包含当前正在执行的函数。MDN

白话：arguments这个实参列表所对应的函数。

caller

caller顾名思义，代表调用者，为函数的属性；

白话：返回当前被调用函数的函数引用，即谁在调用当前函数。

注意：

  1. caller为函数的属性，代表调用该函数的调用者；若该函数在全局作用域，则function.caller为null；而callee为arguments的属性，代表当前正在执行的函数。
  2. caller callee arguments为非标准特性，在严格模式下会报错！
